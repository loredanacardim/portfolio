P r o j e c t s

As a front-end developer, my current focus of study is on designing and building engaging and user-friendly web experiences. I am continually honing my skills in HTML, CSS, and JavaScript to create responsive and optimized web pages that deliver a seamless user experience. I am always seeking to learn new tools and techniques to enhance my abilities and stay at the forefront of the rapidly evolving front-end development landscape. My goal is to create websites and web applications that not only meet but exceed user expectations and drive results for my clients.

While my programming journey began with learning C++ and I have a vast knowledge of the language, it is not my current focus. Nonetheless, my experience with C++ has given me a solid understanding of programming concepts, data structures, and algorithms, which I apply to my front-end development work to create robust, efficient, and effective web applications.
